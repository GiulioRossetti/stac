Descripción

Los tests paramétricos son aquellos que cumplen las condiciones establecidas para los tests paramétricos:

  • Normalidad: Las muestras siguen una distribución normal.
  • Independencia: Las muestras no dependen de otras.
  • Homocedasticidad: Igualdad de varianzas en las muestras.
Ir a Condiciones paramétricas

En las siguientes pestañas "Anova" y "T-test" puedes encontrar los diferentes tests no paramétricos implementados en la web.

Test de Anova

Anova: Contrasta la hipótesis nula de que las medias de los resultados de dos o más algoritmos son iguales, es decir si la media de dos o más muestras o poblaciones son iguales. Para ello, analiza la variación entre muestras y la variación en el interior de las mismas mediante la determinación de varianzas. El estadístico de la prueba de Anova, se aproxima mediante la distribución f, y se rechazará la hipótesis nula si el p-valor < nivel de significación (alpha), aceptando la hipótesis alternativa de que al menos dos medias poblacionales son diferentes.

Se deben cumplir las condiciones paramétricas:

  • Normalidad
  • Homocedasticidad
  • Independencia

Bonferroni: Una vez se tenga constancia gracias al análisis de varianzas Anova de que existen diferencias significativas entre las medias de los algoritmos, se procede con el test Post-Hoc de Bonferroni para determinar las diferencias entre todas las muestras, comparando las medias de todos los algoritmos entre sí. Cada p-valor asociado con la hipótesis Hi se compara con un alpha ajustado a todas las comparaciones: pi < (alpha/m), donde m se corresponde con el número de comparaciones: m=(K∗(K−1))2 y K es el número de algoritmos.

T-test

Si p_valor < alpha, se rechaza la hipótesis nula de que 2 muestras relacionadas o repetidas tienen idénticos valores promedio (esperados). La prueba mide si la puntuación media difiere significativamente entre las muestras. p-valores pequeños están asociados a altos estadísticos t.

Se deben cumplir las condiciones paramétricas:

  • Normalidad
  • Homocedasticidad
  • Independencia

También se dede cumplir que el número de algoritmos o muestras sea de dos.
