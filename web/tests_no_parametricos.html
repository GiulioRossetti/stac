Descripción

Los tests no paramétricos son aquellos que no cumplen alguna de las condiciones establecidas para los tests paramétricos:

  • Normalidad: Las muestras siguen una distribución normal.
  • Independencia: Las muestras no dependen de otras.
  • Homocedasticidad: Igualdad de varianzas en las muestras.

Test de Wilcoxon

Aprender más acerca del test de Wilcoxon.

Nivel de significancia alpha:  ?



Información general:

Hipótesis nula (H0): Las medianas de las diferencias de dos muestras (resultados de dos algoritmos) relacionadas son iguales.

Restricciones:

  • Número de algortimos >= 2
  • Se deben tener al menos 5 conjuntos de datos cuyas diferencias sean distintas de 0.

Test de ranking:  ?

Nivel de significancia alpha:  ?


Maximización o minimización:  ?

Información general:

Hipótesis nula (H0): No existen diferencias entre los grupos o algoritmos.

Comentarios:

  • Friedman: Comparaciones intra-conjunto.
  • Iman-davenport: Comportamiento menos conservativo que Friedman.
  • Rangos Alineados de Friedman: Comparaciones entre conjuntos de datos. Cuando el número de algoritmos a comparar pequeño.
  • Quade: Los rankings se escalan dependiendo de las diferencias observadas en los rendimientos de los algoritmos.
Aprender más acerca de los tests de ranking.

Test Post-Hoc:  ?





Información general:

Hipótesis nulas (H0): No existe diferencia entre el "Algoritmo A" y el "Algoritmo B".

Comentarios:

  • Para los tests Post-Hoc normales, se compara el mejor algoritmo del ranking con los demás.
  • Para los tests Post-Hoc multitests, se comparan todos los algoritmos con todos, con lo que habrá más hipótesis a contrastar.
  • Los tests están ordenados de menor a mayor potencia (tanto normales como multitests).
