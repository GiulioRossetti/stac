Tests Paramétricos
==================

Para realizar contrastes sobre datos cuyas características sean:

* *Independencia*
* *Normalidad*
* *Homocedasticidad*

podemos utilizar los siguientes tests paramétricos:

* :mod:`Test de Anova <tests_parametricos.anova_test>`
* :mod:`Test de Bonferroni <tests_parametricos.bonferroni_test>`
