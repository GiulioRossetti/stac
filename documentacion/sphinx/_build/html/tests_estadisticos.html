Tests Estadísticos¶

En la plataforma se pueden ejecutar dos tipos principales de tests estadísticos: los tests paramétricos y los tests no paramétricos. Los tests paramétricos son aquellos que cumplen las características de:

  • Independencia
  • Normalidad
  • Homocedasticidad

Si alguna de estas condiciones no se cumple, podemos hablar de los tests no paramétricos, que seguirán una distribución no normal como por ejemplo la distribución Chi Cuadrado.

Además, se incluyen tests de criterios paramétricos para comprobar las condiciones de Normalidad y Homocedasticidad.

Requerimientos:

  • Versión de Python: 2.7.6
  • Librerías:
    • SciPy (http://www.scipy.org)
    • NumPy (http://www.numpy.org)
