API REST¶

Los tests de la plataforma así como la subida y consulta de ficheros son accesibles mediante servicios REST.

La plataforma web consta de dos tipos principales de servicios web REST: los servicios de subida y consulta de ficheros y servicios de acceso y ejecución de los tests.

Requerimientos:

  • Versión de Python: 2.7.6
  • Framework Web: Bottle (http://bottlepy.org)
